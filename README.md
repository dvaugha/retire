# RetireSafe

A secure, mobile-first retirement planner that keeps your data 100% local on your device.

## Features
- **Age Window**: Track your timeline from today to retirement.
- **Financial Vault**: Secure local storage for 401k, IRA, and other assets.
- **Runway Projection**: Know exactly when your funds will last.
- **Lifestyle Multiplier**: See the impact of saving a little more each month.

## Security
This app uses **Local Storage Only**. No data is ever sent to a server. Your financial privacy is guaranteed by design.
